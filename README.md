## Project Setup

### Initial setup

1. Clone the site
2. ```sudo gem install bundler```
3. ```bundle install```
4. ```sudo gem install jekyll```

### Build and run

```bundle exec jekyll serve```

```bundle exec jekyll serve --drafts```
